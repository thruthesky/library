# 이용안내

## 접속 경로

/library 로 접속을 하면, 모든 기능을 이용 할 수 있다.


## 회원 정보


* /member/register - 회원 가입
* /member/login - 로그인
* /user/logout - 로그아웃
* /member/update - 회원 정보 수정
